## 0. Окружение и запуск

### Требования
- Python 3 или любой HTTP-сервер (Node, Live Server)
- Браузер Chrome / Firefox / Safari


## 1. Как работают режимы

В приложении два режима — **Клиент** и **Мастер**. Переключение в браузере — кнопками `[Клиент]` `[Мастер]` в верхней части страницы (элемент `#mode-switcher`).

### Логика определения режима (из `js/app.js`):

| Условие | Режим |
|---------|-------|
| start_param === `admin` | Мастер |
| user.id === `123456789` (из `data.js: master.telegram_id`) | Мастер |
| Браузер (нет Telegram SDK) | Клиент (user.id = 999888777 ≠ 123456789) |
| Переключатель #mode-master нажат | Мастер (демо) |

### Поведение при старте в зависимости от флагов localStorage:

| localStorage | Что открывается |
|---|---|
| `welcome_shown` отсутствует | welcome → catalog |
| `welcome_shown = '1'` | catalog |
| Режим мастер, `master.onboarding_done = true` | dashboard |
| Режим мастер, `master.onboarding_done = false` | onboarding |

> В `data.js` значение `onboarding_done: true` — по умолчанию дашборд, не онбординг.

### Сброс состояния между тестами

```javascript
// В консоли браузера:
localStorage.clear();
location.reload();
```

## 5. Браузерные фоллбэки (нет Telegram SDK)

В браузере Telegram SDK (`window.Telegram.WebApp`) недоступен. Код обрабатывает это явно.

| Функция | Поведение в браузере |
|---------|---------------------|
| `getUser()` | Возвращает `{id: 999888777, first_name: 'Тест', last_name: 'Пользователь'}` |
| `MainButton` | Не показывается. В каждом экране есть HTML-кнопка как резервная |
| `BackButton` | Не показывается (навигация — через кнопки на экране или переключатель) |
| `HapticFeedback` | Вызовы игнорируются (optional chaining `?.`) |
| `showConfirm()` | Использует нативный `window.confirm()` |
| `showAlert()` | Использует нативный `window.alert()` |

## 8. LocalStorage — ключи

| Ключ | Значение | Когда записывается | Эффект |
|------|----------|-------------------|--------|
| `offer_shown` | `'1'` | При закрытии оффера | Оффер больше не показывается |
| `welcome_shown` | `'1'` | При клике "Начать" | Welcome-экран больше не показывается |

Для сброса:
```javascript
localStorage.removeItem('offer_shown');
localStorage.removeItem('welcome_shown');
// или полностью:
localStorage.clear();
```
