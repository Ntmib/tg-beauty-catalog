# Telegram Mini App — Каталог бьюти-мастера

> Прототип v1.0 | HTML + CSS + JS (без фреймворков)

## Описание

Telegram Mini App для бьюти-мастеров: каталог услуг, портфолио работ, онлайн-запись.
Два режима: **клиент** (просматривает каталог) и **мастер** (управляет каталогом).

## Запуск

Открыть `index.html` в браузере или через локальный сервер:

```bash
# Простой способ
cd tg-app && python3 -m http.server 8080
# Затем открыть http://localhost:8080

# Или через npx
npx serve tg-app
```

В Telegram — задеплоить на HTTPS и подключить через BotFather → Menu Button.

## Навигация между экранами

### Клиент (каталог)

```
К1 (каталог) → К3 (услуга) → К4 (запись) → К5 (успех)
К1 → К2 (фото)
К1 → К6 (мои записи)
К6 → К4 (повторить/перенести)
```

### Мастер (админка)

```
М0.1 (онбординг, 3 шага) → М1 (дашборд)

М1 → М2 (услуги) → М2.1 (редактирование)
М1 → М3 (портфолио)
М1 → М4 (расписание)
М1 → М5 (записи)
М1 → М6 (профиль)
М1 → К1 (предпросмотр как клиент)
```

## Дизайн-система

### Цвета
Все цвета берутся из Telegram ThemeParams (CSS-переменные `--tg-theme-*`).
В браузере — фоллбэки. Хардкоженных цветов нет.

### Типографика
- Шрифт: системный стек (`-apple-system, SF Pro Text, system-ui`)
- Заголовок: 20px, semibold
- Текст: 15px
- Подпись: 13px, hint color

### Компоненты
- Карточки: `border-radius: 12px`, `background: secondary-bg`
- Chips: `border-radius: 20px`, 36px высота
- Кнопки: `min-height: 44px` (тап-таргет)
- Слоты времени: сетка 3 в ряд, 44px высота

### Анимации
- Переход между экранами: `translateX + opacity`, 250ms
- Появление карточек: `fadeInUp` с каскадной задержкой
- Галочка успеха: `scale` + SVG stroke animation

## Telegram SDK

Используемые методы (через `telegram.js`):
- `expand()` — полный экран
- `disableVerticalSwipes()` — скролл без закрытия
- `MainButton` — основная кнопка действия
- `BackButton` — кнопка "Назад"
- `HapticFeedback` — вибрация при действиях
- `showConfirm()` — нативные диалоги
- `enableClosingConfirmation()` — защита от случайного закрытия
- `requestWriteAccess()` — разрешение на уведомления

## Демо-режим

Переключатель "Клиент / Мастер" вверху экрана.
В Telegram — режим определяется автоматически по `initData.user.id`.
